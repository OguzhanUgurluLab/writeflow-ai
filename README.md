# 🚀 WriteFlow AI - Landing Page

A modern, fully-animated SaaS landing page for an AI writing assistant.

![WriteFlow AI](https://writeflow-ai-phi.vercel.app/)

## 🌐 Live Demo

**[View Live Site →](https://writeflow-ai-phi.vercel.app/)**

## ✨ Features

- 🎨 Modern gradient design with glassmorphism effects
- ⚡ Smooth animations with Framer Motion
- 📱 Fully responsive (mobile-first design)
- 🎯 Conversion-optimized layout
- 🚀 Built with Next.js 14 App Router
- 💎 TypeScript for type safety
- 🎭 Tailwind CSS for styling

## 🛠️ Tech Stack

- **Framework:** Next.js 14
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Deployment:** Vercel

## 🚀 Getting Started

```bash
# Clone the repo
git clone https://github.com/KULLANICI_ADIN/writeflow-ai.git

# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
